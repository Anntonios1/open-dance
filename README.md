# Open Dance

Un simulador interactivo de Just Dance en el navegador.

## Características
- Menú de inicio y selección de canciones
- Selección de coach
- Gameplay con pictogramas, puntuación y feedback visual
- Pantalla de resultados

## Requisitos
- Node.js 18 o superior
- npm

## Instalación
```bash
npm install
```

## Ejecución
```bash
npm start
```

La app estará disponible en:
```text
http://localhost:3000
```

## Estructura principal
- `public/` — interfaz y juego en navegador
- `server.js` — servidor Express
- `package.json` — dependencias y scripts

## Nota
La carpeta `MAPS-HIGHHD/` está ignorada en el repositorio por tamaño.
